## Needs Attention Section
- Shows cattle with stress level вүҘ moderate
- Sorted: danger вҶ’ severe вҶ’ moderate
- Uses same card component as Cattle List
- Tap card вҶ’ Cattle Detail screen
- If all healthy: show "рҹҺү All cattle are healthy!" message
